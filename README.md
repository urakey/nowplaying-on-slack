# nowplaying-on-slack

いま聴いている曲をSlackのStatusに表示させる Last.fm 版

## Start

```
% SLACK_API_TOKEN=xoxp-xxxx-xxxx-xxxx-xxxx \
  LASTFM_API_KEY=xxxxxxxxxxxxxxxxxxx \
  LASTFM_SHARED_SECRET=xxxxxxxxxxxxxxxxxxx \
  LASTFM_USER_NAME='xxxxx' \
  npm start
```
